# PackagePlugin
可视日历控件实现了日期范围限制、两个日期控件范围联动等功能。鼠标悬停可以显示农历。

# 软件简介 

工作中很多项目用到jquery的日期控件，功能很强大，使用也方便，所以就以学习及巩固java基础为目的写了个java版的日历控。实现了日期范围限制、两个日期控件范围联动等功能。鼠标悬停可以显示农历。


开发环境：jdk6+

开发工具：netbeans 7.0
